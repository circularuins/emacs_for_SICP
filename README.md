## インストール
$ cd ~/  
$ mv .emacs.d .emacs.d_old  
$ git clone https://github.com/circularuins/emacs_for_SICP.git .emacs.d  

## 起動
$ emacs

## 基本コマンド
* ファイルを開く：C-x C-f
* ファイルを保存：C-x C-s
* バッファの切替：C-x b
* バッファを閉じる：C-x k
* ウィンドウの分割（水平）：C-x 2
* ウィンドウの分割（垂直）：C-x 3
* ウィンドウの消去（自分以外）：C-x 1
* ウィンドウの選択：C-x o
* emacsの終了：C-x C-c
